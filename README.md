# 7Wonders

C++ and Qt multiplayer implementation of 7Wonders by Antoine Bauza.

Tested on Windows 10 and Arch Linux (Qt 5.15.0 gcc 10.1.0 64 bits).

Player hosting the game needs a public IP, or should use a VPN.
